# 开发思路 #

## 目录规范 ##
ZoomQuiet: 将一个版本的所有文件聚合在一齐，形成规范，以后各个版本统一使用:
```
eb0
+-- army            ~ 指挥脚本
+-- ebin            ~ 编译成果
+-- conf            ~ 配置文件
+-- core            ~ 核心引擎
+-- test            ~ 测试案例
+-- util            ~ 工具代码
+-- i18n            ~ 国际化
+-- erlbattle.erl   ~ 启动脚本
+-- configure       ~ 配置引导
+-- README          ~ 使用指南
`-- INSTALL         ~ 安装指南
```
