# 路线图:Road Map #

## 概述 ##
路线图`包含`作品/项目各个阶段的核心目标简述, `为`开发人员 `在`配置管理/发布管理/变更管理等等各种软件工程关键过程中 `提供` 了最基础的命名和开发方向指导作用;
  * 并且`作为`团队内部沟通的基础隐喻, `设计`了唯一的独特名字来指代各种功能集合!

`Road Map ~ 就是当前作品/项目/计划变化趋势的脉络图`;必须每个阶段有个贴切和简单的名字对应!作为配置中核心的概念和统一使用的工作对照引述标识！
