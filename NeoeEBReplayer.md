# Introduction #
另一个battlelog播放器，把warfield.txt转成一组图片进行浏览。

# Details #

EbReplay 里的播放器比较华丽。 这个比较简陋，但是便于快速浏览。

运行前提：
需要安装JRE5.0+

运行方法：
  * 1. 下载 http://erlbattle.googlecode.com/files/ebrep.7z
  * 2. 解压到_你的目录_
  * 3. 把要看的warfield.txt拷贝的_你的目录_
  * 4. 运行run.cmd
  * 5. 程序会生成一组PNG图片文件到_你的目录_/warpic/
  * 6. 浏览
